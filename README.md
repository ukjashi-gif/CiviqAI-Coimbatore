## 📌 Overview

CivicAI Coimbatore is an intelligent Emergency Dispatch & Incident Command System designed to help city authorities monitor, manage, and respond to emergencies in real time.

The platform provides a centralized dashboard for incident tracking, GIS visualization, emergency reporting, AI-assisted decision support, analytics, and automated SITREP (Situation Report) generation.

This project was built as a Smart City emergency management solution to demonstrate how AI and real-time data can improve disaster response, resource allocation, and operational efficiency.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/civiqai-coimbatore.git
```

## Install Dependencies

```bash

```

## Run Development Server

```bash

```

Runs on

